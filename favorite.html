Git 기본 명령어

Git에서 사용되는 기본 명령어를 알아볼까요?

  • git init - 새로운 Git 저장소 생성
  • git add - 변경된 파일을 스테이징
  • git commit -m "메시지" - 변경 사항을 기록
  • git push - 원격 저장소에 변경 사항 업로드

Git 협업 흐름

팀 프로젝트에서 Git을 활용하여 협업을 해 봅시다!

  1. GitHub에서 레포지토리를 생성하고 `git clone`으로 가져오기
  2. 새로운 기능을 개발할 때 브랜치를 생성하고 작업하기
  3. 수정한 내용을 `git add`, `git commit`으로 저장
  4. `git push` 후 Pull Request 생성하여 코드 리뷰 요청
  5. 리뷰가 끝나면 `git merge` 후 원격 저장소에 반영
